# Blabb - Social Networking Application in C++

Blabb is a C++ social networking app where users can create profiles, add friends, make posts, like & comment, and manage pages.

## Project Structure
- src/ → C++ source files
- include/ → Header files

## Features
- User Sign Up / Login
- Profile Management
- Friend Requests & Management
- Post Creation, Likes, and Comments
- Page Creation & Management
- Memory Sharing
- News Feed (Friends & Pages)
- File-based Data Storage
- GUI using raylib / raygui
